DLFlow-Cookiecutter-V1
======================

Golden Flow Maker for DL. 

Requirements
------------
Install `cookiecutter` command line: `pip install cookiecutter`    

Usage
-----
Generate a new Cookiecutter template layout: `cookiecutter gh:ravivijay/DLFlow-Cookiecutter-V1`
